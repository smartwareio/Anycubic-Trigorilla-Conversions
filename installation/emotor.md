# How to wire the E Motor

From the **Swap Board** to the **SKR 3 EZ**:

- Run a cable from E1A to E0M 1A
- Run a cable from E1B to E0M 1B
- Run a cable from E2A to E0M 2A
- Run a cable from E2B to E0M 2B
