# How to wire the X Motor

From the **Swap Board** to the **SKR 3 EZ**:

- Run a cable from X1A to XM 1A
- Run a cable from X1B to XM 1B
- Run a cable from X2A to XM 2A
- Run a cable from X2B to XM 2B
