# How to wire the extruder thermistor

From the **Swap Board** to the **SKR 3 EZ**:

- Run a wire from T0 GND to TH0 GND on the SKR 3 EZ.
- Run a wire from T0 I/O to TH0 PA2 on the SKR 3 EZ.
