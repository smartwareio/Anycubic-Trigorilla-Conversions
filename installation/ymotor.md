# How to wire the Y Motor

From the **Swap Board** to the **SKR 3 EZ**:

- Run a cable from Y1A to YM 1A
- Run a cable from Y1B to YM 1B
- Run a cable from Y2A to YM 2A
- Run a cable from Y2B to YM 2B
